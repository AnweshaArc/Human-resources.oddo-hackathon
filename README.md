# Sleep Environment & Wellness Tracker

## Description
This project aims to identify the underlying causes of sleep disturbances, insomnia, and morning tiredness by analyzing the relationship between environmental conditions and sleep quality. Room temperature and humidity data is continuously collected using sensors and pushed to Firebase in real-time, while users manually log how they feel during and after sleep (e.g., restlessness, discomfort, quality of rest). By correlating environmental sensor data with subjective sleep feedback, the system helps uncover patterns behind poor sleep and morning fatigue, enabling users to make informed adjustments to their sleep environment for better rest.

## Features
- Real-time temperature and humidity monitoring via sensors
- Sensor data logged and stored on Firebase
- Manual sleep quality and feeling entries by the user
- Correlation analysis between environmental conditions and sleep quality
- Insights into possible causes of insomnia and morning tiredness
