About Sleep & Wellness Tracker

Sleep & Wellness Tracker is an IoT-based smart sleep monitoring system designed to help users understand how their surrounding environment influences sleep quality. The dashboard continuously visualizes room temperature, humidity and sleep statistics, making it easier to identify healthy sleep patterns.

Project Overview

This project combines IoT sensors with a modern web dashboard to provide real-time monitoring and insightful analytics for better sleep health.

Key Features

  • Real-Time Temperature Monitoring
  • Real-Time Humidity Monitoring
  • Sleep Quality Dashboard
  • Interactive Analytics Charts
  • Sleep History Records
  • Sleep Report Logging
  • Responsive User Interface
  • Help & Support Widget

Technology Stack

  • HTML5
  • CSS3
  • JavaScript
  • Chart.js
  • Firebase (Sensor Storage)
  • ESP32 / IoT Sensors

Future Scope

  • AI Sleep Recommendations
  • Machine Learning Predictions
  • Mobile Application
  • Wearable Device Integration
  • Smart Home Automation
  • Cloud Analytics

Developer

Sleep & Wellness Tracker Team

This project was developed as a hackathon solution to demonstrate how IoT technology and web analytics can be combined to improve sleep quality. The system focuses on providing an intuitive dashboard, live monitoring, historical analysis and actionable insights for healthier sleep habits.
